# Site évènementiel de Pâques 

 ## Objectifs
  - Faire un site évènementiel pour la fête de Pâques.
  - Attirer l'attention des personnes sur des fêtes religieux.
  - Porter le site à évoluer et parler d'autre fêtes que Pâques.
